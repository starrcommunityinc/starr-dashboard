# STARR Community Services International
## Volunteer Impact Dashboard

Live dashboard tracking volunteer hours, program reach, and community impact for STARR Community Services International, Inc.

**Live dashboard:** https://starrcommunityinc.github.io/starr-dashboard

---

## What This Is

This repository hosts the public-facing impact dashboard for STARR Community Services International. The dashboard displays volunteer hours, participants served, program value, and leverage ratio — all drawn from STARR's Google Sheets volunteer tracker.

No personal data is stored here. All figures are aggregate totals only.

---

## Files

| File | Purpose |
|---|---|
| `index.html` | The dashboard — opens in any browser |
| `dashboard_data.json` | Exported from Google Sheets after each event — no PII |

---

## How to Update the Dashboard After an Event

1. Open the STARR Volunteer Impact Tracker in Google Drive
2. Click **⭐ STARR Tools** → **📊 Export Dashboard JSON**
3. Download `dashboard_data.json` from Google Drive
4. Come back to this repository → click `dashboard_data.json` → click the pencil icon (Edit) → replace the contents → click **Commit changes**
5. The live dashboard updates automatically within 1–2 minutes
